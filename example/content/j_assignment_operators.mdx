## JS Assignment Operators

### Markdown table

| Operator  |   Example    |  Same as       |
|-----------|:------------:|---------------:|
| `=`       |   `x = y`    | `x = y`        |
| `+=`      |   `x += y`   | `x = x + y`    |
| `-=`      |   `x -= y`   | `x = x - y`    |
| `*=`      |   `x *= y`   | `x = x * y`    |
| `/=`      |   `x /= y`   | `x = x / y`    |
| `%=`      |   `x %= y`   | `x = x % y`    |
| `<<=`     |   `x <<= y`  | `x = x << y`   |
| `>>=`     |   `x >>= y`  | `x = x >> y`   |
| `>>>=`    |   `x >>>= y` | `x = x >>> y`  |
| `&=`      |   `x &= y`   | `x = x & y`    |
| `^=`      |   `x ^= y`   | `x = x ^ y`    |
| `\|=`     |   `x \|= y`  | `x = x \| y`   |
| `**=`     |   `x **= y`  | `x = x ** y`   |

`\` added before `|` symbol to display it in a table.

## Assignment Examples

The `=` assignment operator assigns a value to a variable.

**`=` Assigment**

```js
var x = 10;
```

The `+=` assignment operator adds a value to a variable.

**`+=` Assignment**

```js
var x = 10;
x += 5;
> console.log(x)
15
```

The `-=` assignment operator substracts a value from a variable.

**`-=` Assignment**

```js
var x = 10;
x -= 5;
> console.log(x)
5
```

The `/=` assignment divides a variable.

**`/=` Assigment**

```js
var x = 10;
x /= 5;
> console.log(x)
2
```

The `%=` assignment operator assigns a remainder to a variable.

**`%=` Assigment**

```js
var x = 10;
x %= 5;
> console.log(x)
0
```
