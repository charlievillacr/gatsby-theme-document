## The HTML script element is used to embed or reference executable code; this is typically used to embed or refer to JavaScript code. The script element can also be used with other languages, such as WebGL's GLSL shader programming language.

```js 
<!DOCTYPE html>
<head>
    <script>function laFuncion() {document.getElementById("demo").innerHTML="El párrafo cambió!";}</script>
</head>
<html>
    <body>
        <h2></h2>
        <p id="demo">Un párrafo.</p>
        <button type="button" onclick="laFuncion()">Click me</button>
    </body>
</html>
```

### Script acepta los siguientes atributos: async, crossorigin, defer, integrity, nomodule, nonce, referrerpolicy, src, type.
